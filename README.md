# GameOn — Landing Page with JavaScript

A landing page featuring a dynamic registration form built with Vanilla JavaScript, as part of a front-end development training program.

🔗 **[Live Demo](https://gwen-herissant.github.io/GwenaelleHerissant_4_23122021/)**

## Project Overview

This project was completed as part of a structured front-end development curriculum. Working for a fictional company that organizes gaming contests, the goal was to implement an interactive registration form in JavaScript on top of an existing HTML/CSS codebase.

It served as an introduction to JavaScript and the dynamic interactions between HTML, CSS, and JS to build functional, user-friendly web pages.

## Objectives

- Build and validate a form using Vanilla JavaScript
- Handle user events (clicks, inputs) to create dynamic interactions
- Work within an existing codebase (HTML & CSS provided)

## Key Features

- **Form validation** — client-side validation ensuring user inputs meet defined requirements before submission
- **Event handling** — dynamic interactions triggered by user actions such as button clicks and form inputs
- **Vanilla JS** — no libraries or frameworks, pure JavaScript fundamentals

## What I Learned

- Writing JavaScript from scratch without frameworks or libraries
- Validating form data and providing user feedback
- Handling DOM events to create interactive experiences
- Understanding how HTML, CSS, and JavaScript work together

## Context

This project is part of the **OpenClassrooms Front-End Developer** curriculum. It marks the introduction to JavaScript in the program, focusing on core programming fundamentals and DOM manipulation.
